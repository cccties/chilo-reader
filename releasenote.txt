New features
• Added function to divide the video to each page as described in the SMIL file
• Added function to turn the page automatically

Specification changes
• Play the video automatically when a page is turned
• Hide the zoom menu icon on the navigation bar
• Changed the size of the logo button in the navigation bar
• Deleted the title attribute value of the logo image
• Added class attribute value to the buttons in the navigation bar 
• Added ".htaccess" file to "reader_script" directory
